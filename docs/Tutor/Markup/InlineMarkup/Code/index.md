#### Synopsis

Create inline code fragments.

#### Syntax

``````
`MarkedText`
``````

#### Description

Creates inline fragments of text that represent code.
Inside the _MarkedText_, other markup can be used.

#### Examples

* `` `if` ``  gives `if`.
* to escape backquotes simply surround by more: ``` `` `if` `` ```
