.Synopsis
Definition of the Rascal standard prelude.

.Description

The standard prelude is automatically imported when Rascal is started.
It combines the following modules:

*  `Boolean`,
*  `DateTime`,
*  `Exception`,
*  `IO`,
*  `List`,
*  `Map`,
*  `Node`,
*  `ParseTree`,
*  `Rational`, ???
*  `Relation`,
*  `Set`,
*  `String`,
*  `ToString`,
*  `ValueIO`.
