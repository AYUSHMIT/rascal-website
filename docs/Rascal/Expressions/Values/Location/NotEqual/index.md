#### Synopsis

The not equal operator on location values.

#### Syntax

`Exp<sub>1</sub> != Exp<sub>2</sub>`

#### Types

| `Exp<sub>1</sub>` | `Exp<sub>2</sub>` | `Exp<sub>1</sub> != Exp<sub>2</sub>`  |
| --- | --- | --- |
| `loc`     |  `loc`    | `bool`                |


#### Function

#### Description

Yields `true` if both arguments are not identical and `false` otherwise.
