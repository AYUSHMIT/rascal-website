
[[Set-NotEqual]]
# Set NotEqual
:concept: Expressions/Values/Set/NotEqual

.Synopsis
Not equal operator on set values.

.Syntax
`_Exp_~1~ != _Exp_~2~`

.Types


|====
| `_Exp~1~_`    |  `_Exp~2~_`    | `_Exp~1~_ != _Exp~2~_` 

| `set[_T~1~_]` |  `set[_T~2~_]` | `bool`               
|====

.Function

.Description
Yields `true` if both arguments are unequal sets and `false` otherwise.

.Examples
[source,rascal-shell]
----
rascal>{1, 2, 3} != {3, 2, 1};
bool: false
rascal>{1, 2, 3} != {1, 2};
bool: true
----
