# Boolean Negation

.Synopsis
The _not_ operator on Boolean values.

.Index
!

.Syntax
`! Exp`

.Types

//

| `Exp` | `! Exp`  |
| --- | --- |
| `bool`   | `bool`     |


.Function

.Details

.Description
The _not_ operator on Boolean values defined as follows:

| `Exp`  | `! Exp`  |
| --- | --- |
| `true`   | `false`    |
| `false`  | `true`     |


.Examples
```rascal-shell
!true;
```
